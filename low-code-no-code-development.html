06 Delivered Mocks (released in Anypoint Code Builder)

Deliver, Launch and Moving forward

After I defined, designed and delivered the Flow Building and other core experiences for API and Integration development, and sprints of collaberation with engineer team and product team, Anypoint Code Builder was initially released in Open Beta around February 2023. This initial beta version was a zero-install IDE that ran entirely in your browser, aiming to eliminate the need for local software downloads and machine spec considerations.

Anypoint Code Builder Desktop achieved General Availability (GA) on October 16, 2023. This desktop version is built on Visual Studio Code, offering a familiar and modern IDE experience with MuleSoft extensions for visual building and testing of integrations.

Our team has continued to release updates and new features for Anypoint Code Builder since its GA launch, with releases in February, June, and October of 2024, and ongoing updates in 2025.
